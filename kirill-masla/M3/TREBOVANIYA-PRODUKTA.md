# TREBOVANIYA-PRODUKTA — Требования к продукту (PDR)
**Модуль:** M3 — Продуктовая стратегия
**Клиент:** Кирилл Бродский (Магия Масел)
**Дата:** 30.03.2026
**Версия:** 1.0

## 6.1. Что продукт ДОЛЖЕН иметь

### Производство
- **Гидравлический пресс** (не шнековый) — ключевой дифференциатор по качеству отжима
- **Температура ≤ 40°C** — зафиксированная датчиком, не задекларированная
- **Малые партии** — 40–60 л/мес, отжим под спрос, не под склад
- **Тёмное стекло** — защита от ультрафиолета, обязательно для всей линейки

### Документация и доверие
- **Государственные сертификаты** на каждое масло
- **Сертификаты сырья** от поставщика (не только на готовый продукт)
- **Дата отжима** на каждой бутылке — не дата розлива и не срок годности
- **Дозатор** — контроль дозировки, часть сервиса

### Продуктовая логика
- **Моно-масла** — только чистые масла, без смешивания
- **Карточка применения** в каждом наборе — как применять, зачем, кому
- **Инструкция «30-дневный протокол»** в наборах 3×100 мл — конкретная схема применения
- **Напоминание на повтор** — встроено в сервис (полуручной ремайндер)

### Коммуникация
- **Честное объяснение** — что масло делает, что не делает, кому противопоказано
- **Предупреждение о противопоказаниях** — всегда, без исключений
- **Кирилл как лицо** — голос бренда присутствует в каждом касании

## 6.2. Что продукт НЕ должен иметь

| Что запрещено | Причина |
|---------------|---------|
| Купажи / смеси масел | Кирилл категорически против. Нет технологических карт |
| Капсульная форма | Отказ подтверждён однозначно |
| 500 мл для премиальных масел | Нет смысла — только лён и подсолнечник |
| 50 мл | Тары нет |
| Обещания лечения | Юридические и этические ограничения |
| Медицинские заявления | «Инструмент, не чудо-средство» |
| Агрессивные CTA | «Купи сейчас!», «Последний шанс» — противоречат голосу бренда |
| Амарант и облепиха (пищевые) | Нереализуемо — нужен пресс 100 т |
| Маркетплейсы | Не рассматриваются на текущем этапе |
| Молодёжный сленг | Не соответствует аудитории и тону бренда |

## 6.3. Минимальные стандарты качества

### Сырьё
- Семена из проверенных хозяйств с датой уборки
- Свежесть сырья — приоритет над ценой

### Процесс
- Контроль температуры на прессе — датчик, не оценка на глаз
- Фильтрация без химии — только механика
- Разлив в тёмное стекло сразу после отжима

### Готовый продукт
- Каждая партия — под спрос (не залёживается на складе)
- Срок хранения соответствует дате отжима (не максимально растянутый)
- Этикетка содержит дату производства конкретной партии

### Документация
- Госсертификаты актуальны
- Сертификаты сырья хранятся и могут быть показаны

## 6.4. Сервисные стандарты

### До покупки
- Кирилл лично объясняет: какое масло → для какой задачи → как принимать
- Консультация без давления — «попробуйте, если понравится — приходите»
- Для онлайн: подбор через TG или квиз/инструкция

### После покупки
- **Полуручной ремайндер:** система напоминает Кириллу → он пишет клиенту лично
  - 100 мл: напоминание на 21–25 день
  - 250 мл: напоминание на 28–30 день
  - 500 мл: напоминание на 45–50 день
- Повторная продажа через личный контакт, не массовую рассылку

### Набор / подарок
- Подарочный бокс: Кирилл собирает вручную → персональный характер
- Карточка применения вложена в каждый набор
- Упаковка доставки: вакуумная (уже применяется)

### Гарантия (онлайн)
- Возврат если вкус не подошёл
- 100% возврат если данные не соответствуют заявленным
