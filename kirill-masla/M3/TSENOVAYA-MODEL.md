## 6. ОБОСНОВАНИЕ ЦЕНЫ (как объяснять покупателю)

Цена = сумма доказательств. Каждый элемент должен быть виден или объяснён:

| Элемент | Что значит | Статус |
|---------|------------|--------|
| Гидравлический пресс (не шнек) | Нет нагрева → нет окисления → полезные вещества сохранены | ✅ Готов объяснять |
| Тёмное стекло | Защита от ультрафиолета | ✅ Уже в упаковке |
| Малые партии | Масло свежее, не залёживается | ✅ Реальность производства |
| Госсертификат | Официальная проверка | ✅ Есть на все масла |
| Видео производства | "Вижу кто и как делает" | ⚠️ Нужно снять |
| Дата отжима | "Знаю когда сделано" | ⚠️ Нужно добавить на этикетку |
| Сертификат сырья | Поставщик проверен | ⚠️ Запросить у поставщика |
| Инструкция по применению | "Знаю как использовать" | 🔶 Есть для 3 масел, нужно расширить |

## 7. 🔴 ЦЕНОВОЙ РИСК — НИЖЕ РЫНКА

> Кирилл продаёт на 10–15% дешевле конкурентов. При заявленном качестве выше рынка — это когнитивный диссонанс.

**Проблема:** "Хорошее дешевле хорошего" — не работает как логика в премиум-категории. Покупатель недоверчив.

**Рекомендация M3:**
- Не снижать цены дальше
- При первом онлайн-запуске — протестировать рыночные цены (не ниже среднего)
- Если Кирилл хочет быть "чуть дешевле" — это позиция, которую нужно объяснять: "Я не переплачиваю за маркетинг и аренду офиса — только производство"
- Альтернатива: специальная цена для подписчиков TG = нет публичного демпинга
