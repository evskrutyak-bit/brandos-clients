## Методология

Из 18 конкурентов Артефакта 1 отбираем 5 для глубокого досье. Критерий — не «кто самый большой», а **кто наиболее релевантен для принятия тактических решений по проекту Кирилла**.

### Критерии скоринга

| Критерий | Вес | Что оцениваем |
|----------|-----|---------------|
| **Пересечение по ЦА** | 25% | Их покупатель ≈ покупатель Кирилла (ЗОЖ 30–60, средний+ доход, осознанность) |
| **Пересечение по продукту** | 25% | Линейка пересекается: те же масла, тот же формат, та же ценовая зона |
| **Пересечение по каналам** | 15% | Конкурируют в тех же каналах (D2C, Telegram, Instagram, маркетплейсы, рынок) |
| **Сила digital** | 20% | Развитость маркетинга, соцсетей, контента, воронки |
| **Чему учиться** | 15% | Уникальные практики, форматы, механики, которые стоит изучить |

Баллы: 1 (минимум) — 5 (максимум) по каждому критерию.

## Расчёт (для воспроизводимости)

Формула: `Итого = ЦА×0.25 + Продукт×0.25 + Каналы×0.15 + Digital×0.20 + Учиться×0.15`

**Проверка TRAWA:** 5×0.25 + 5×0.25 + 4×0.15 + 5×0.20 + 5×0.15 = 1.25 + 1.25 + 0.60 + 1.00 + 0.75 = **4.85** ✓

**Проверка Радость Здоровья:** 5×0.25 + 5×0.25 + 5×0.15 + 3×0.20 + 4×0.15 = 1.25 + 1.25 + 0.75 + 0.60 + 0.60 = **4.45**

*Корректировка: при пересчёте Радость Здоровья = 4.45, не 4.35. Итоговый рейтинг остаётся тем же — 2-е место.*

## Обоснование отбора

### Топ-5: почему именно они

**1. TRAWA (4.85)** — безальтернативный выбор. Единственный системный игрок в премиальном D2C. Задаёт ценовой потолок ниши, формирует ожидания ЦА. Всё, что Кирилл строит в digital, будет сравниваться с TRAWA. Максимум данных в источниках — можно построить полноценное досье. Кроме того, из досье TRAWA извлекаются уязвимости (нет Telegram, нет квиза, нет подписки), которые напрямую влияют на стратегию Кирилла.

**2. Радость Здоровья (4.45)** — ближайший структурный аналог Кирилла. Тоже Москва, тоже точка на рынке, тоже сыродавленное масло, тоже D2C через мессенджеры, тоже премиум. Это зеркало: что у них работает — может работать у Кирилла. Где они слабы — Кирилл может быть сильнее. Максимальное пересечение по каналам (5 баллов — единственный конкурент, который продаёт через рынок + мессенджер + сайт).

**3. Altaria (3.80)** — представитель инновационного сегмента. Не прямой конкурент по продукту (спрей ≠ стекло), но по ЦА и каналам — пересекается. Важен потому что: (а) показывает как работает инфлюенс-маркетинг в масляной нише, (б) 10 000+ отзывов — механика, которую стоит изучить, (в) формат Consumer Experience — тренд, к которому нужно быть готовым.

**4. Дом Кедра (3.15)** — представитель маркетплейс-сегмента. Самый широкий ассортимент (23 масла), агрессивное дисконтирование, уникальная линейка BIO-hepa (капсулы масло+экстракт). Важен для понимания: как маркетплейсы формируют ценовые ожидания, какие продуктовые форматы востребованы, как работает скидочная модель. Если Кирилл выйдет на Ozon/WB — это прямой конкурент по полке.

**5. Масляный Король (2.80)** — антипример. Попал в топ-5 не по итоговому баллу (формально 7-е место после Маслослада и Пазла Здоровья), а по стратегической значимости: 22+ года на рынке, Гран-при ПродЭкспо, широчайшая дистрибуция — но полный провал в digital. Это ценнейший кейс: как не надо. Что происходит с сильным продуктом без маркетинга. Кроме того, его цены (льняное 260 ₽/250 мл) — это ценовой якорь, с которым Кирилл будет бороться в сознании покупателя.

**Замена по скорингу:** формально 5-е место занимает Маслослад (3.20) и Пазл Здоровья (2.85). Масляный Король (2.80) включён вместо Маслослада по причине стратегической значимости — ценовой якорь + антипример digital-провала важнее для проекта, чем ещё один крафтовый D2C. Эта замена прозрачна и обоснована.

### Кого «почти взяли»

**6-й — Маслослад (3.20).** Структурный аналог (Москва, крафт, D2C, дубовый пресс), но: (а) слишком похож на «Радость Здоровья», которая уже в топ-5; (б) данных в источниках мало — профиль получится тонким; (в) меньше уникальных механик для изучения (YouTube-канал — единственная фишка).

**7-й — Пазл Здоровья (2.85).** Интересен уникальным позиционированием (масла-миксы под задачу, Telegram-напоминания), но: (а) другой продукт (миксы, не моно-масла); (б) пограничные медицинские обещания — Кирилл не может копировать этот подход; (в) данных мало.
