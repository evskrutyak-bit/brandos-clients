## Блок 3. Кластеризация отзывов

### Распределение по темам (оценка на основе собранных данных)

| Кластер | Доля (оценка) | Что говорят | Тональность |
|---------|--------------|-------------|-------------|
| **Вкус / запах** | ~35% | «вкусное», «не горчит», «горькое», «нежный аромат», «сбалансированный вкус» | Преимущественно позитив |
| **Свежесть** | ~20% | «свежее», «не самого свежего отжима», «произведено в ноябре» | Позитив + единичные жалобы |
| **Упаковка** | ~15% | «упаковано хорошо», «тщательно запакованное», «тёмное стекло» | Позитив |
| **Формат / удобство** | ~10% | «нравится спрей», «быстро окисляется в бутылках» | Позитив (Altaria) |
| **Цена** | ~10% | Имплицитно — на масла TRAWA не жалуются на цену (премиальная аудитория). На Масляный Король — жалоб тоже нет (масс-маркет) | Нейтрально |
| **Доверие / подлинность** | ~5% | «никогда не знаешь, холодного отжима или разбавленное» | Негатив (скепсис) |
| **Доставка** | ~5% | Упоминается в контексте «пришло быстро», но не доминирует | Нейтрально |

### Language Mining: слова клиентов

**Позитивный лексикон:**
- «Свежее», «вкусное», «натуральное», «настоящее», «не горчит»
- «Тщательно запакованное», «упаковано хорошо»
- «Заказала сразу ещё» (триггер повторной покупки)
- «Рекомендую!»

**Негативный лексикон:**
- «Горькое» (льняное — типичная проблема окисления)
- «Не самого свежего отжима» (разрыв обещания)
- «Никогда не знаешь» (скепсис к подлинности)
- «Разбавленное» (страх фальсификата)

**Нейтральные паттерны:**
- «Полезное масло для салата» (утилитарное восприятие)
- «Масло очень понравилось» (общее без деталей)

## Блок 4. Незакрытые потребности

### Что клиенты хотят, но не получают

| Потребность | Откуда видно | Кто мог бы закрыть |
|-------------|-------------|-------------------|
| **Гарантия подлинности / свежести** | «Никогда не знаешь, холодного отжима или разбавленное» | Кирилл: лабораторные анализы + видео отжима |
| **Прозрачность процесса** | «Не самого свежего отжима» — ожидание vs реальность | Кирилл: дата отжима на бутылке, видео конкретной партии |
| **Защита от окисления** | «Быстро окисляется в бутылках» | Altaria решает спреем. Кирилл: азотное заполнение, мелкая фасовка, инструкция по хранению |
| **Помощь с выбором** | Отзывы не сравнивают масла между собой. Покупатель не знает, что выбрать | Кирилл: квиз-подборщик, гайд «масло под задачу» |
| **Объяснение горечи** | «Горькое» — частая жалоба на льняное масло | Кирилл: образовательный контент «Почему льняное горчит и когда это нормально» |
| **Персональный контакт** | Ответы брендов шаблонные | Кирилл: личные ответы от производителя |
| **Мини-формат на пробу** | Покупатели заказывают большие объёмы, не попробовав | Кирилл: триальный набор 3 × 100 мл |
| **Рецепты / применение** | «Полезное масло для салата» — клиенты не знают, как ещё использовать | Кирилл: вкладыш с рецептами / QR-код на карточку с рецептами |

## Блок 5. Рекомендации для Кирилла

### Язык клиентов → офферы

| Слова клиентов | Оффер для Кирилла |
|----------------|-------------------|
| «Свежее», «настоящее» | → «Дата отжима на каждой бутылке. Видео вашей партии в Telegram» |
| «Не горчит» | → «Льняное масло, которое не горчит — потому что отжато сегодня» |
| «Никогда не знаешь» | → «Лабораторный анализ каждой партии — QR-код на этикетке» |
| «Тщательно запакованное» | → «Тёмное стекло + крафт-упаковка + инструкция по хранению» |
| «Заказала сразу ещё» | → «Подписка: свежее масло каждый месяц без повторного заказа» |
| «Нравится спрей» | → Не копировать Altaria, но учесть: мелкая дозировка, удобный дозатор |

### Контент из отзывов

| Тема контента | Формат | Источник инсайта |
|--------------|--------|-----------------|
| «Почему льняное масло горчит — и когда это нормально» | TG-пост + статья | Жалобы на горечь у Масляного Короля |
| «Как проверить, что масло настоящее ХО» | Reels + TG | Скепсис «никогда не знаешь» |
| «Что значит "отжим после заказа" и почему это важно» | TG-пост | Жалоба на TRAWA «не самого свежего» |
| «5 способов использовать льняное масло кроме салата» | Карусель IG | «Полезное масло для салата» — утилитарность |
| «Как хранить масло ХО, чтобы оно не испортилось» | TG + Дзен | «Быстро окисляется в бутылках» |
| Видео: «Отжимаю масло под ваш заказ — вот процесс» | Reels / Shorts | Потребность в прозрачности |

### Система работы с отзывами (с момента запуска продаж)

1. **Каждый отзыв — личный ответ от Кирилла** (не шаблон). Преимущество: ни один конкурент не делает это персонально.
2. **Негативные отзывы → контент.** Каждая жалоба = тема для поста/Reels.
3. **Позитивные отзывы → UGC.** Репост в TG/IG с разрешения. TRAWA делает это через @trawa_recept — можно масштабировать.
4. **NPS после каждого заказа** (TG-бот: «Оцените от 1 до 10»). Данные → улучшение продукта и контента.
