02 — Профили по конкурентам

Детальный разбор каждого игрока

TRAWA (trawaoil.ru) — оценка 4/5

Instagram (@trawa_oil) — основной канал

  • 95K подписчиков, 1284 публикации [веб, 03.04.2026]
  • Второй аккаунт: @trawa_recept (рецепты от подписчиков)
  • Описание профиля: «Сыродавленные масла на дубовых бочках · Сладости БЕЗ глютена, сахара · Соусы и суперфуды»
  • Форматы: Reels, фото продукции, рецепты, партнёрский контент
  • Темы: рецепты с маслами, процесс производства, партнёры (ГУМ, Radisson), акции, сезонные предложения
  • Активность: регулярная (свежие публикации фиксируются в поиске)

VK / Telegram / YouTube

  • VK (vk.com/trawaoil): 6 549 подписчиков, 1 464 записи. Дублирует контент из Instagram с адаптацией. Менее активная аудитория. [веб, 03.04.2026]
  • Telegram (t.me/trawa_oil): Канал-бот: ссылка на заказ + полезности + другие соцсети. Не является самостоятельным контент-каналом — скорее точка конверсии.
  • YouTube (TRAWA): ~729 подписчиков. Контент: Shorts (производство, отжим масла), обзорные видео. Просмотры низкие (100-150 на видео). Не является приоритетным. [веб, 03.04.2026]

Единственный, кто системно ведёт соцсети. Слабые места: YouTube не развит, Telegram — утилитарный, нет TikTok.

Altaria (altariaoil.ru) — оценка 3/5

Instagram (@altaria.oil) — основной канал

  • ~187K подписчиков, 6.7M суммарных просмотров Reels [веб, 03.04.2026]
  • Описание: «Масла в формате спрея — must have на вашей кухне. Чат в TG с персональным менеджером»
  • Форматы: Reels (готовка со спреем, рецепты, лайфхаки), фото продукции
  • Темы: кулинарные рецепты, экономия калорий через спрей, подача как ЗОЖ-инструмент
  • Региональные аккаунты: @altaria_oil_56 (Оренбург)
  • Визуал: яркий, food-oriented, акцент на «тонкую готовку»

VK: Присутствует, описание: «Масла для здоровья в каждый дом и на любой вкус»

Telegram: Не классический канал, а TG-чат с персональным менеджером. Модель: персональная продажа через мессенджер.

Блог на сайте: Публикации с 2021 по 2024 — рецепты, уход за кожей маслами, восточная кухня, выставки (WorldFood Moscow 2024). Нерегулярный (паузы в несколько месяцев).

Парадокс: 187K в Instagram — больше, чем у TRAWA, но бренд не транслирует это в SEO или в другие каналы. Instagram работает за счёт формата (спрей = визуально привлекательно для Reels).

Дом Кедра (dom-kedra.ru) — оценка 2/5
  • Instagram: Не обнаружен для «Дом Кедра» масло. Возможно, нет или ведётся без привязки к бренду. [веб, 03.04.2026]
  • VK (vk.com/domkedransk): Присутствует, метрики не извлечены. Ссылка на сайте, вероятно B2B/лояльная аудитория.
  • Telegram (t.me/dom_kedra): Канал для оптовиков [веб, 03.04.2026]. Не D2C, не контент-канал.
  • YouTube (@domkedra): Есть канал, контент: производственный блог [веб, 03.04.2026]. Аудитория не зафиксирована, просмотры предположительно низкие.
  • Одноклассники, WhatsApp: Есть ссылки на сайте. ОК — специфическая аудитория 40+. WhatsApp — точка заказа, не контент.

Присутствие формальное, контента для конечного потребителя нет. Все каналы заточены под B2B/опт. Нет Instagram — критический пробел для D2C-бренда.

Масляный Король (maslo53.ru) — оценка 1/5
  • VK (vk.com/maslo53vn): Присутствует. Описание: «21 вид натуральных растительных масел». Активность: предположительно низкая (B2B-ориентация). [веб, 03.04.2026]
  • Facebook (facebook.com/maslo53.ru): Страница существует. Facebook в РФ заблокирован — канал нефункционален. [веб, 03.04.2026]
  • Instagram, Telegram, YouTube: Не обнаружены. [веб, 03.04.2026]

Минимальное присутствие. VK — формальная страница. Facebook — бессмысленный в РФ. Нет Instagram, нет Telegram, нет YouTube. Бренд не работает с конечным потребителем через соцсети.

Радость Здоровья (radostzdorovia.ru) — оценка 0/5
  • Не обнаружено присутствие ни на одной платформе [веб, 03.04.2026]
  • Сайт не содержит ссылок на соцсети
  • Единственный способ связи: телефон +7 (926) 134-42-61

Полное отсутствие в соцсетях. Микробизнес без digital-стратегии. Ликвидированное юрлицо.

TRAWA — эталон контент-стратегии в нише
  • Единственный, кто ведёт UGC-аккаунт (@trawa_recept)
  • Системная работа с рецептами — и на сайте, и в IG
  • Партнёрский контент (ГУМ, Radisson, Бахетле) — даёт социальное доказательство

  • Контент крутится вокруг одного USP: спрей = экономия = тонкая готовка
  • Не используют потенциал аудитории для других каналов
  • Блог на сайте заброшен (последние активные посты — 2024)

Ни один конкурент не делает: личный бренд основателя · образовательный контент о маслах (польза, применение, наука) · сравнения масел (какое выбрать для задачи) · истории клиентов / трансформации · контент «от сырья до бутылки» · TikTok / Дзен · системный email-маркетинг

Стратегический вывод

Платная реклама в digital для D2C-масел практически отсутствует у всех пяти конкурентов. Это подтверждает вывод из Артефакта 3 (SEO): ниша не освоена performance-маркетингом.

Единственные значимые промо-активности:

  • TRAWA — партнёрства с премиальными площадками (ГУМ, Radisson) вместо прямой рекламы
  • Altaria — выставки (WorldFood Moscow) и блогерские интеграции для B2B-развития
  • Altaria — возможный таргет в Instagram (при 187K подписчиков органический рост маловероятен без платного промо, но подтвердить нельзя)
05 — Контентные разрывы

Матрица: что никто не делает

РазрывОписаниеПриоритет для Кирилла
Личный бренд основателя Ни один конкурент не строит бренд через личность. TRAWA = корпоративный бренд, Altaria = продуктовый бренд. Нет лица. Критический — прямое преимущество Кирилла
Образовательный контент «Как выбрать масло», «чем отличается сыродавленное», «как хранить» — никто не делает системно Критический — связка с SEO-стратегией (Артефакт 3)
Telegram как контент-канал TRAWA = бот для заказа. Altaria = чат с менеджером. Никто не ведёт Telegram как полноценный канал Высокий — основная платформа
TikTok / Reels о производстве Altaria показывает готовку, TRAWA — рецепты. Но никто не показывает отжим «от ореха до бутылки» глазами производителя Высокий — визуально мощный формат
Сравнения и гайды «Какое масло для чего», «5 масел на каждый день», «масло для печени vs для кожи» — контентная пустота Высокий
Истории клиентов Ни один конкурент не публикует трансформации / результаты клиентов Средний — требует базы клиентов
Дзен Ни один конкурент не использует Яндекс.Дзен. Бесплатный канал с SEO-эффектом Средний
Email-маркетинг Не обнаружен ни у одного конкурента для D2C Средний — на этапе когда будет база
06 — Рекомендации для Кирилла

Приоритетные платформы и контент-стратегия

Ранжирование платформ

ПлатформаПриоритетПочему
Telegram№1 — КритическийОсновная платформа. Ни один конкурент не ведёт полноценный канал. Прямая связь с аудиторией, нет алгоритмов.
Instagram№2 — Критический95K у TRAWA, 187K у Altaria — аудитория есть. Reels = основной формат роста. Личный бренд + производство = уникальный контент.
YouTube Shorts№3 — ВысокийПроцесс отжима, «ореховое масло за 60 секунд», образование. TRAWA набирает только 150 просмотров — легко обойти.
Дзен№4 — СреднийБесплатный, SEO-эффект, длинные статьи. Никто из конкурентов не использует.
VK№5 — НизкийРепост из других каналов. Не приоритет, но присутствие нужно.

Контент-стратегия: рубрики

РубрикаФорматЧастотаПример
ПроизводствоReels / Shorts / TG2/нед«Отжимаю кедровое масло под ваш заказ. Смотрите процесс»
ОбразованиеКарусели IG / TG-посты / Дзен2/нед«Чем сыродавленное отличается от холодного отжима — объясняю как производитель»
РецептыReels / TG1/нед«3 салата с льняным маслом, которые вы будете делать каждую неделю»
Личный брендStories / TG / Reels1-2/нед«Почему я ушёл из [прошлая деятельность] и начал давить масло»
СравненияКарусели / TG / Дзен1/нед«Льняное vs оливковое: что полезнее и для чего»
КлиентыTG / IG1/2 нед«Ольга заказывает кедровое масло каждый месяц. Вот её история»
Целевая частота: 7-10 единиц контента в неделю
  • Telegram: 5 постов/неделю (основной канал, короткий формат + длинные лонгриды)
  • Instagram: 3-4 поста/неделю (Reels + карусели)
  • YouTube Shorts: 1-2/неделю (репост Reels + уникальный контент)
  • Дзен: 1 статья/неделю (образовательный лонгрид, SEO-эффект)
  • VK: репост из TG/IG

Ключевое отличие от конкурентов: TRAWA продаёт продукт. Altaria продаёт формат. Кирилл должен продавать экспертизу + процесс + личность. Ни один конкурент не использует формулу: производитель-эксперт с лицом → образование → доверие → покупка. Это и есть стратегическое окно.
