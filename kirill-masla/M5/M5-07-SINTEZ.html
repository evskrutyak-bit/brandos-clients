01 — Ключевые находки модуля

6 блоков исследования: итоги

1.1. Рынок

Рынок масел холодного отжима в России — $862 млн (2024), CAGR 4,7% до 2030. Растёт за счёт clean label, специализированных масел (кедровое, тмин, тыква) и проникновения ХО в массовые категории.

Три ценовых яруса: Эконом (130–400₽/250 мл), Средний (400–900₽/250 мл), Премиум (900–3060₽/250 мл).

Кирилл — в премиальном ярусе. Прямой конкурент — TRAWA. Ценовое столкновение неизбежно только в случае копирования их модели. Задача — не конкурировать с TRAWA по цене, а строить другую ценность.

1.2. Конкурентное поле

Исследовано 18 конкурентов (Артефакт 1), 5 — глубоко (профили + SEO + соцсети + офферы + отзывы).

Главный вывод: рынок digital-слеп. Из 18 конкурентов только TRAWA ведёт системный digital-маркетинг. Altaria — одноканальный Instagram-бренд. Остальные 16 — либо B2B-оптовики, либо маркетплейс-зависимые, либо вообще без присутствия.

Карта позиционирования 2×2: Q2 (дорого + сильный digital) — только TRAWA. Квадрант практически пуст. Текущий Кирилл — Q4 (дорого + слабый digital). Задача проекта: перевести из Q4 в Q2.

1.3. SEO
  • 4 из 5 топ-конкурентов невидимы в поиске. TRAWA и Дом Кедра — 3/5 по SEO-зрелости, Altaria и Масляный Король — 2/5, Радость Здоровья — 1/5. Ни один не достиг уровня 4/5.
  • 70% семантического ядра — информационные запросы, которые не закрывает ни один производитель. Топ забит мед. порталами и ЗОЖ-блогами. Производитель с экспертным контентом = E-E-A-T-преимущество по умолчанию.
  • Платная реклама в нише = ноль. Ни один из 5 конкурентов не использует контекстную рекламу. Аукцион пуст.
1.4. Соцсети и контент

Только два конкурента создают контент: TRAWA (95K IG, 6.5K VK, YouTube) и Altaria (187K IG). Остальные три — ноль.

Ни один конкурент не делает:

  • Личный бренд основателя
  • Образовательный контент о маслах
  • Сравнения «какое масло для чего»
  • Telegram как контент-канал
  • TikTok / Дзен
  • Email-маркетинг
  • Истории клиентов
1.5. Офферы и воронки

Собрано 42 дословных оффера (Артефакт 5). Доминирующие триггеры: технология, авторитет, свежесть. Единственный по-настоящему уникальный оффер — TRAWA: «Начинаем отжим только после поступления заказа».

Ни один конкурент не использует: квиз-подборщик · подписку / автодоставку · триальный набор · email-воронку · гарантию возврата · лид-магниты для D2C.

Оценки воронок: TRAWA 3/5, Altaria 2/5, Дом Кедра 1.5/5, Масляный Король 1/5, Радость Здоровья 0.5/5.

1.6. Голос клиента

Позитивный лексикон: «свежее», «вкусное», «настоящее», «не горчит», «тщательно запакованное».

Негативный лексикон: «горькое», «не самого свежего отжима», «никогда не знаешь», «разбавленное».

Главная неудовлетворённая потребность: доверие и прозрачность. Покупатели хотят доказательств — что масло действительно свежее, действительно ХО, действительно не разбавлено. Ни один бренд не даёт объективных proof-точек (анализы, видео конкретной партии, дата отжима на бутылке).

04 — Незанятые ниши и стратегические окна

10 окон по приоритету захвата

Критические (занять первым — структурное преимущество)

1. Telegram как премиальный контент-канал масел
Из 18 конкурентов ни один не ведёт Telegram для D2C-аудитории. TRAWA = бот для заказа. Altaria = чат с менеджером. «Пазл Здоровья» = напоминания. Полноценный канал с образованием + личностью + продажами = открытое окно. ЦА Кирилла (30–60, ЗОЖ) активна именно в TG.
2. Личный бренд производителя масел
Нулевой конкурент использует формулу «лицо + экспертиза + процесс». TRAWA — корпоративный. Altaria — продуктовый. Остальные — безликие. Кирилл как «масляный эксперт с лицом» = уникальная позиция в нише, не копируемая быстро.
3. Proof-маркетинг (лабораторные анализы + видео партии)
Потребители говорят: «никогда не знаешь, холодного отжима или разбавленное». Ни один бренд не даёт объективных доказательств. QR-код на этикетке → анализ партии + видео отжима = закрытие главной боли рынка.

Высокие (быстрый захват, низкая конкуренция)

4. Квиз-подборщик масла
Не реализован ни одним конкурентом. Покупатели не знают, что выбрать (подтверждено отзывами). 7 вопросов → рекомендация → промокод = лид-магнит + сегментация + конверсия. Уже запланирован в M7-GTM.
5. Образовательный SEO-контент
70% семантики ниши — информационные запросы. В топе — мед. порталы и ЗОЖ-блоги. Ноль производителей. «Что такое сыродавленное масло» — в топе budetpolza.ru одной статьёй. Производитель с E-E-A-T = автоматическое преимущество.
6. Подписка «Масло месяца»
Бутылка 250 мл расходуется за 3–4 недели. Масло — продукт регулярного потребления, но продаётся как разовая покупка. Ни один из 18 конкурентов не предлагает автодоставку. Подписка = LTV × 3–5, предсказуемая выручка, снижение CAC.
7. Триальный набор 3–5 × 100 мл
Формат 100 мл есть (TRAWA, Siberian Green), но дегустационных сетов 5+ вкусов нет. Входной продукт с низким порогом + повод для повторной покупки полноразмерных бутылок.

Средние (занять на этапе масштабирования)

8. Наборы «под цель»
TRAWA делает миксы (масла в одной бутылке), но не кураторские наборы из разных масел под сценарий. Средний чек Кирилла офлайн (3–4 тыс.₽ = несколько бутылок) подтверждает спрос.
9. Контент о противопоказаниях и честности
Все бренды обещают пользу. Никто не строит позиционирование на честности: «мы предупреждаем, не обещаем чудес». Антиоффер сильнее «живого эликсира» в контексте скепсиса к ЗОЖ-маркетингу.
10. YouTube Shorts / Reels о производстве
TRAWA показывает рецепты, Altaria — готовку со спреем. Никто не показывает отжим «от ореха до бутылки» глазами производителя. TRAWA YouTube — 729 подписчиков, 100–150 просмотров. Легко обойти.
05 — Стратегия дифференциации

3 оси, TOWS-матрица, Blue Ocean

TRAWA продаёт продукт. Altaria продаёт формат. Кирилл должен продавать экспертизу + процесс + личность.

Три оси дифференциации

Ось 1: Личность

«Кирилл Бродский — производитель, который знает каждый орех в вашей бутылке.»

Ни один конкурент не может скопировать быстро. Личный бренд = долгосрочный барьер входа.

Ось 2: Прозрачность

«Дата отжима на каждой бутылке. Лабораторный анализ по QR-коду. Видео вашей партии в Telegram.»

Закрывает главную боль рынка: «никогда не знаешь». Объективные proof-точки vs субъективные заявления.

Ось 3: Образование

«Не просто масло — система знаний о том, как его выбрать, использовать и хранить.»

70% семантики ниши — информационные запросы без ответа от производителей. Кирилл = эксперт, который учит.

08 — Что передаётся в M6-позиционирование

Подтверждённые факты и конкретные решения

Подтверждённые факты (не гипотезы)

  • TRAWA — единственный системный digital-конкурент в премиуме. Остальные 17 — либо не в digital, либо не в премиуме.
  • Личный бренд производителя = незанятая позиция. 18 конкурентов, ноль использует формулу «лицо + экспертиза + процесс».
  • Telegram для D2C масел = пустое окно. Ни один из 18 конкурентов.
  • 70% семантики ниши не закрыто производителями. Информационные запросы обслуживают мед. порталы, не бренды.
  • Главная боль покупателей — доверие. «Никогда не знаешь, настоящее или нет.» Proof-маркетинг = прямой ответ.
  • Платная реклама в digital = ноль. Аукцион пуст. Даже минимальный бюджет даст непропорциональный результат.
  • Ни один конкурент не использует квиз, подписку, триальный набор, email-воронку.

Гипотезы для проверки в M6

#ГипотезаОснованиеКак проверить
1 Кирилл может занять Q2 (премиум + сильный digital) за 6 месяцев Конкурентная пустота в квадранте Требуется регулярный контент (7–10 единиц/неделю)
2 Ценовая позиция «ниже TRAWA, выше среднего» (60–75% от цен TRAWA) — оптимальна на старте Анализ ценовых ярусов Проверка эластичности спроса на триальном наборе
3 Образовательный контент = основной канал органического роста 70% семантики не закрыто производителями ROI измерить через 3–6 месяцев после запуска
4 Формула «экспертиза + процесс + личность» сильнее, чем «технология + премия + бренд» (TRAWA) Отсутствие личных брендов у всех конкурентов A/B-тест на реальной аудитории
