3. Позиционирование v0 — первые формулы

Формула «для кого / что / против кого / почему» + 3 угла A/B/C + УТП + архетип.

3.1. Формула позиционирования v0

Для эстетов-домохозяек 27–45 лет и их семей,
которые ценят дизайн интерьера и воспринимают кошку как члена семьи,
Alien Cat — дизайнерский трон из натурального дуба и эко-меха,
который делает кошкины потребности невидимой частью интерьера,
в отличие от когтеточек-зоотоваров (уродуют интерьер) и китайского масс-маркета (ДСП + сизаль + плохой клей),
потому что только мы сочетаем органическую форму + авторский почерк + slow-living нарратив + цену 35–70k.

3.2. Альтернативные углы — A / B / C

Три возможных «главных» угла — в M6 выберем один. Рабочая гипотеза: угол A (арт-объект) с нюансами B (нарратив slow-living) для контент-слоя. Угол C держим в резерве для B2B-коммуникаций с дизайнерами.
